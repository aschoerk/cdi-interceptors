com.github.sfleiter.cdi_interceptors

Annotation Type Logging


  • @InterceptorBinding
@Target(value={METHOD,TYPE})
@Retention(value=RUNTIME)
public @interface Logging
    Activates automatic method-level logging for any annotated bean. For every call a single slf4j log message is generated with the logger set to the name of the class. The message contains the method name, the parameters, the result object, the call duration and the exception if one got thrown.
    Author:
    Stefan Fleiter
